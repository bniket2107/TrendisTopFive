1. Setup Python Environment: Before running the project, ensure that Python is installed on your system. You will also need a few Python libraries for this project. Use the following steps to set up the environment: 
Install Python (if not already installed) from python.org. Install the required Python libraries. You can do this by running the following commands in your terminal/command prompt:
pip install selenium requests webdriver-manager
2. Install WebDriver: The project uses the Chrome WebDriver to control the browser. The webdriver-manager package automatically installs the correct version of ChromeDriver, so no manual download is required.
